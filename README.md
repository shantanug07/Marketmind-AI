# 📈 MarketMind-AI

AI-Powered Stock Market Intelligence Platform

## 🚀 Features

- Download live stock market data
- Supports US and Indian stocks
- Saves data as CSV
- Performs basic data analysis
- Generates summary statistics
- Detects missing values

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- yFinance
- Matplotlib
- Plotly
- Streamlit
- Scikit-Learn
